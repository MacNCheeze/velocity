FileCopier Class

Defined in: lib/FileCopier.js:9
Module: Velocity

Worker that copies test files to the mirror reactively.

Constructor

FileCopier

(
  • options
)

Defined in lib/FileCopier.js:9

Parameters:

  • options Object
    • targetFramework String

      The name of the target framework for which the tests should be copied to the mirror.

    • [onFileAdded] Function optional

      Callback that is called after a file was added.

    • [onFileChanged] Function optional

      Callback that is called after a file has changed.

    • [onFileRemoved] Function optional

      Callback that is called after a file was removed.

    • [shouldCopy] Function optional

      Control whether a file is copied. Passed the file object from the VelocityTestFiles collection which has a absolutePath field. Default: true

    • [convertTestPathToMirrorPath] Function optional

      A function that converts the test file path to the mirror path.

Example:

var fileCopier = new Velocity.FileCopier({
  targetFramework: TEST_FRAMEWORK_NAME
});
fileCopier.start();
